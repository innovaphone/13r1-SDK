ethernet

class IEthernet

static bool GetMACAddress(byte * mac, class IInstanceLog * const log)
Writes the MAC address of the first ethernet adapter to the given * mac array. Returns true if a MAC address was found, otherwise false. log is used to log debugs dependent of the log flag LOG_ETHERNET. The byte array is not NULL terminated and just contains the six MAC bytes!

Code Example 

byte mac[MAC_ADDRESS_SIZE];
bool success = IEthernet::GetMACAddress(mac, this); // this is an AppInstance object
