Protocols implemented by the innovaphone PBX and Apps

Overview

There are various protocols used by the innovaphone PBX and Apps to communicate. These protocols are mostly based on JSON messages transmitted over a Websocket connection

AppWebsocket
This is the basis of most communication between Apps and the PBX or App Services
Signal
This is the generic JSON signaling signaling protocol, which is mapped by the PBX to any standard Voip Signaling like SIP or H.323
PbxSignal
Protocol to register to the PBX and do signaling with the JSON signaling protocol
EpSignal
This protocol allows to attach to a device registered to the innovaphone PBX and intercept all signaling to and from this device
Messages
The protocol to retrieve stored chat messages
myApps
The protocol the myApps client uses to communicate with the PBX
