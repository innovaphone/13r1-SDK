certificates

class ICertificates

static certificate_error_t ValidateCertificate(class IInstanceLog * log, const byte * cert, size_t certLen)
byte * cert must contain a certificate in PEM format. Other formats are currently not supported.
ValidateCertificate checks if there is a public key, a private key and verifies the certificate chain.

Code Example 

byte * cert ...;
size_t certLen ...;
certificate_error_t error = ICertificates::ValidateCertificate(this, cert, certLen);
